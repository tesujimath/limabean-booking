# limabean-booking

This is a generic implementation of the Beancount booking algorithm, with no dependencies on either the [Lima parser](https://github.com/tesujimath/beancount-parser-lima) nor [limabean](https://github.com/tesujimath/limabean) itself.

For further information see the [API documentation](https://docs.rs/limabean-booking/latest/limabean_booking/).
